Idle jGrowl Rerouting

Using Paul Irish's idleTimer plugin, it is possible to queue up Notifications like jGrowls. The "sticky" setting sets the jGrowl sticky setting to true. The push setting sends all the Growls to a Notification Centre style modal.

The idleTimer is triggered after 2500ms of inactivity, and jGrowls are triggered every 5000ms.
