bootstrap.footnotes.js

Example HTML

Using a number fallback is good accessibility, and resembles normal footnotes/endnotes. 

<sup data-text="This is an aside">1</sup>

JavaScript Invokation

// default syntax, searches for "sup" element
$.footnotes();

// optionally specify a selector
$.footnotes(".foo");
